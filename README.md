# Thoughts in Bloom

A personal digital garden for capturing, connecting, and reflecting on your thoughts. Built as a quiet, contemplative space where ideas can grow over time.

## What it does

- **Capture thoughts** as they come — from observations, conversations, books, dreams, or questions
- **Tag and connect** related ideas to see how your thinking evolves
- **Organize by seasons** — group thoughts into meaningful periods of your life
- **Constellation view** — visualize how your thoughts connect through shared tags
- **Insights and analytics** — discover patterns in your thinking over time
- **Archive** — set aside thoughts you've moved past without losing them

## Privacy

All data is stored locally in your browser using localStorage. Nothing is sent to a server — your thoughts stay entirely on your device.

## Built with

- [Next.js](https://nextjs.org) (App Router)
- [React](https://react.dev)
- [TypeScript](https://www.typescriptlang.org)
- [Tailwind CSS](https://tailwindcss.com)
- [D3.js](https://d3js.org) for the constellation visualization
- [Framer Motion](https://www.framer.com/motion) for animations

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to start your garden.
