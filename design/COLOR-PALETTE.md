# Thoughts in Bloom - Color Palette & Design Tokens

## Color Palette

### Primary Colors (Dreamy Blues/Purples)
| Color Name | Hex Code | Usage |
|------------|----------|-------|
| Primary Blue | `#667eea` | Buttons, accents |
| Primary Purple | `#9f7aea` | Gradients, focus states |
| Light Blue | `#e8f4f8` | Background gradient start |
| Light Purple | `#f0e6f6` | Background gradient end |
| Soft Lavender | `#e6e0f5` | Background gradient end |

### Text Colors
| Color Name | Hex Code | Usage |
|------------|----------|-------|
| Heading | `#4a5568` | Main headings |
| Body | `#2d3748` | Thought card text |
| Subtitle | `#718096` | Tagline, descriptions |
| Muted | `#a0aec0` | Timestamps, placeholders |

### Surface Colors
| Color Name | Hex Code | Usage |
|------------|----------|-------|
| Card Background | `#ffffff` | Thought cards |
| Border Light | `#e2e8f0` | Input borders |
| Shadow | `rgba(0, 0, 0, 0.08)` | Card shadows |
| Shadow Hover | `rgba(0, 0, 0, 0.12)` | Card hover shadows |
| Button Shadow | `rgba(102, 126, 234, 0.4)` | Button hover glow |

## Spacing System

| Token | Value | Usage |
|-------|-------|-------|
| Space XS | `0.5rem` | Tight spacing |
| Space SM | `0.75rem` | Small gaps |
| Space MD | `1rem` | Default spacing |
| Space LG | `1.5rem` | Card padding |
| Space XL | `2rem` | Section margins |

## Border Radius

| Token | Value | Usage |
|-------|-------|-------|
| Radius SM | `8px` | Buttons, inputs |
| Radius MD | `12px` | Cards |
| Radius LG | `16px` | Large elements |

## Typography

- **Font Family**: System fonts (`-apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, sans-serif`)
- **Line Height**: 1.6
- **Heading Size**: Default (browser)
- **Body Size**: 1rem (16px)
- **Small Text**: 0.875rem (14px)
- **Font Weight Regular**: 400
- **Font Weight Bold**: 600

## Animation Timing

| Token | Value | Usage |
|-------|-------|-------|
| Duration Fast | `0.2s` | Hover effects |
| Duration Normal | `0.3s` | Transitions |
| Duration Slow | `0.4s` | Slide-in animations |
| Easing Default | `ease` | General transitions |
| Easing Bounce | `cubic-bezier(0.34, 1.56, 0.64, 1)` | Playful effects |

## Design Principles

1. **Dreamy & Airy**: Soft gradients, generous whitespace, subtle shadows
2. **Clean & Minimal**: No visual clutter, focus on content
3. **Responsive**: Works beautifully on all screen sizes
4. **Animated**: Smooth transitions that feel natural, not distracting
