# Design Assets

This folder contains Figma exports and design documentation for Thoughts in Bloom.

## Contents
- `design-system.md` - Color palette, typography, spacing values
- `figma-links.md` - Links to live Figma files
- Design exports (PNGs) - Visual references for implementation

## Workflow
1. Create designs in Figma
2. Export key screens/components as PNG (2x scale)
3. Document design values in `design-system.md`
4. Reference during CSS implementation
